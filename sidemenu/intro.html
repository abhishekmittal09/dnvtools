Dependency and Version Tracking Tools

The following are some simple use cases of the tool

Dependency graphs - Version Tracking

Visualize annotated dependency graphs at the project and bundle level to understand what project depend on each other and how

Version Skew Information

Track version skew. Understand when different projects or different bundles are pulling in different versions of the same dependency. Get a report which says where this skew exists and periodic warnings for projects that are lagging.

Enhance the Database

Add/Update the database, by running the script periodically.
